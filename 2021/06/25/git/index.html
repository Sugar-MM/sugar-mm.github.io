Jun 25, 2021

reflog 和 git log的区别

git log可以显示所有提交过的版本信息，不包括已经被删除的 commit 记录和 reset 的操作

git reflog显示所有的操作记录，包括提交，回退的操作。一般用来找出操作记录中的版本号，进行回退。

今天就开发中因为提交，回退加切换分支的一系列操作，导致之前版本号用git hist找不到了，然后用git reflog一试，找到了！所以遇到找不回的版本号时，不妨可以试试git reflog
